📑 Expected Extracted Data by Document Type
This guide defines the key fields that each AI agent will extract from documents uploaded under each category in the Documents tab.

Each set of fields will be returned as a structured JSON object and displayed in the UI (e.g. via expandable panels or modals).

🔐 1. Identification Document
Examples: Passport, driving licence, proof of address

json
Copy
Edit
{
  "type": "Identification Document",
  "firstName": "string",
  "surname": "string",
  "dateOfBirth": "YYYY-MM-DD",
  "documentNumber": "string",
  "expiryDate": "YYYY-MM-DD",
  "documentType": "string",        // e.g. "Passport"
  "issuingCountry": "string"       // e.g. "UK"
}
